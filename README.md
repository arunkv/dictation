# Dictation Game

Simple game that helps children practice dictaion.
* Built with OpenAI's TTS API. 
* See https://arunkv.medium.com/the-sound-of-learning-using-openais-text-to-speech-api-for-a-simple-dictation-game-for-kids-e237db497ad1 for more details.

## Required Packages
* `pygame` (https://www.pygame.org/)
* `pyyaml` (https://pyyaml.org/)
* `termcolor` (https://pypi.org/project/termcolor/)
* `openai` (https://pypi.org/project/openai/)

## APIs
* OpenAI's TTS-API (https://beta.openai.com/docs/)
* Needs valid `OPENAI_API_KEY`
